# BusStop Simulation

Prerequisites:
*** Java Development Kit (JDK) 8 or higher must be installed.

Files:
*** BusStop.java: Contains the main simulation logic, rider and bus arrival methods, and concurrency controls.

## How to Run the Program

Step 1: Compile the Program
    1. Open a terminal or command prompt.
    2. Navigate to the directory where the BusStop.java file is located.
    3. Compile the program using the following command: 
        
        javac BusStop.java

Step 2: Run the Program

        java BusStop
